# Leffakerho 🎬

Vaadin-sovellus, jossa käyttäjät voivat selata elokuvia, kirjoittaa arvosteluja, keskustella ja hallita omaa profiiliaan.

## Käynnistys

```bash
./mvnw spring-boot:run

Tai Dockerilla:

docker-compose up --build

Toiminnot

    🔐 Käyttäjätunnukset ja roolit

    🎥 Elokuvien selaus ja suosikit

    📝 Arvostelujen kirjoitus

    💬 Keskustelut

    👤 Profiilin hallinta
